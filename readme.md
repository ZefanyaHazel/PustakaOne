# 📚 Portal Perpustakaan Online

Website Portal Perpustakaan Online **PustakaOne.** adalah aplikasi berbasis web yang memungkinkan pengguna untuk mencari, meminjam, dan memberikan ulasan terhadap buku secara digital ataupun langsung dari perpustakaan terkait. Sistem ini juga menyediakan fitur manajemen bagi admin serta petugas untuk mengelola data buku, kategori, dan peminjaman.

---

## 🚀 Fitur Utama

Terdapat 3 role utama dalam sistem perpustakaan ini, yaitu:
### 👤 User

* 🔍 Mencari buku berdasarkan judul
* 📖 Melihat detail buku (judul, penulis, penerbit, kategori, sinopsis)
* ⭐ Memberikan rating dan ulasan buku
* 📚 Menambahkan buku ke favorit
* 📥 Melakukan peminjaman buku
* 🕒 Memilih durasi peminjaman (7, 14, 30 hari)
* ✏️ Mengedit ulasan yang sudah dibuat

**Username: user**
**Password: user123**

### 🛠️ Petugas & Admin

* ➕ Menambahkan data buku
* ✏️ Mengedit dan menghapus buku
* 🗂️ Mengelola kategori buku
* 📊 Melihat data peminjaman
* ✅ Mengubah status peminjaman (pending, dipinjam, dikembalikan)
* 👤 Mengelola daftar anggota
* 👤 Mengelola daftar petugas (Admin)
* 📊 Melihat Laporan Perpustakaan
